### 网络结构按五层和七层分别是？
1. TCP / IP 体系结构
    * 网络接口层
    * 网际层 IP
    * 运输层 TCP 或 UDP
    * 应用层（TELNET FTP SMTP等）
    * 五层

2. 五层
    * 物理层
    * 数据链路层
    * 网络层
    * 运输层
    * 应用层

3. 七层：Open System Inerconnect Reference Model 开放式系统互联通信参考模型
    * 物理层
    * 数据链路层
    * 网络层
    * 传输层
    * 会话层
    * 表达层
    * 应用层
