## 5. 回文数 (`palindrome-number`)
[LeetCode传送门](https://leetcode-cn.com/problems/palindrome-number/)

题目大意：判断一个整数是否是回文数。

回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

基本知识

跟上面差不多，字符串操作

基本思路
太简单，直接看代码
写法实现

```JavaScript
/**
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function(x) {
    return `${x}`.split('').reverse().join('') === `${x}`
};
```
